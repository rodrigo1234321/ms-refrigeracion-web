Crítico

Error de Certificado SSL / HTTPS en el Dominio en Vivo

El dominio en vivo https://msrefrigeracion.com.ar/ presenta un error de certificado TLS (válido para otro host). Esto bloquea el rastreo de Google y ahuyenta al 100% de los usuarios debido a las advertencias de seguridad del navegador.

Solución: Reinstalar o renovar el certificado SSL (Let's Encrypt gratuito) desde el panel de control del hosting (cPanel, Plesk o Wiroos) para el dominio correcto.

Crítico

Ausencia de Datos Estructurados (Schema Markup)

Ninguna de las páginas posee marcado estructurado JSON-LD. Esto impide que Google interprete de manera enriquecida los servicios técnicos, el tipo de negocio local (LocalBusiness) y los datos de contacto en Mar del Plata.

Solución: Inyectar el siguiente bloque de marcado estructurado en el <head> de index.html:

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "name": "MS Refrigeración",
  "image": "https://msrefrigeracion.com.ar/assets/images/logo.svg",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "Bermejo 758",
    "addressLocality": "Mar del Plata",
    "addressRegion": "Buenos Aires",
    "postalCode": "7600",
    "addressCountry": "AR"
  },
  "telephone": "+542235054977"
}
</script>
Advertencia

Sitemap y Robots.txt Obsoletos de WordPress en el Servidor

El servidor en vivo sigue sirviendo archivos sitemap.xml y robots.txt remanentes de la instalación antigua de WordPress, apuntando a carpetas inexistentes (/wp-admin/). Esto genera desorientación en los robots de búsqueda.

Solución: Crear archivos locales limpios sitemap.xml y robots.txt en la raíz del proyecto para sobreescribir los remanentes.

Advertencia

Ausencia de Etiquetas Open Graph y Twitter Cards

El código actual carece de propiedades meta para redes sociales (og:title, og:image, etc.). Cuando se comparte el enlace en WhatsApp o LinkedIn, no se renderizan imágenes ni descripciones atractivas.

Solución: Agregar meta tags Open Graph básicos en todas las páginas.
